## ✅ 部署后立即验证

### 1. 404 页面
- [ ] 访问 `https://bazu-ranking.com/随便一个不存在的页面`

- [ ] 访问 `https://bazu-ranking.com/robots.txt`
- [ ] ✅ 应显示纯文本，**不应包含 HTML 代码**

### 3. sitemap.xml
- [ ] 访问 `https://bazu-ranking.com/sitemap.xml`

- [ ] ✅ 应显示一张 1200×630 的 SVG 图片

### 5. 默认语言改为英文
- [ ] 打开 `https://bazu-ranking.com`（**未登录过**的浏览器或无痕模式）

### 7. Favicon
- [ ] ✅ 浏览器标签页应显示闪电⚡形状的橙色图标

## 🔍 SEO 提交

### Google Search Console
1. 打开 https://search.google.com/search-console
2. 添加属性：`https://bazu-ranking.com`
3. 验证方式选 **URL prefix** → 用 DNS TXT 记录或 HTML 标签验证
4. 提交 Sitemap：`https://bazu-ranking.com/sitemap.xml`
5. 请求索引：首页 URL

### 百度站长平台
1. 打开 https://ziyuan.baidu.com/
2. 添加站点：`bazu-ranking.com`
3. 验证方式选 **文件验证** → 下载验证文件放到网站根目录
4. 提交 Sitemap：`https://bazu-ranking.com/sitemap.xml`

### Bing Webmaster Tools
1. 打开 https://www.bing.com/webmasters/
2. 添加站点并提交 sitemap

## ⚠️ 注意事项

- **Cloudflare 缓存**：部署后可能需等待 1-5 分钟或手动清除 Cloudflare 缓存才能生效
- **_worker.js 高级模式**：确保 Cloudflare Pages 部署配置为 **Advanced Mode**（使用 `_worker.js`），而不是 Functions 模式
- **环境变量**：确认 `DB` 绑定（D1 数据库）在 Cloudflare Pages 环境中已正确配置
